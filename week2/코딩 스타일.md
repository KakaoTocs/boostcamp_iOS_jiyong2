### switch
- 들여쓰기는 자동으로 적용되는 정도를 사용한다 또는 tab 키를 사용한다.
- case A: 띄어쓰기 사용한다.

```swift
	 func getGrade(average: Double) -> String {
            switch average {
            case 90.0 ... 100.0:
                return "A"
            case 80.0 ... 90.0:
               return "B"
            case 70.0 ... 80.0:
                return "C"
            case 60.0 ... 70.0:
              return "D"
            default:
              return "F"
            }
        }
```
